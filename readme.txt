== Description ==

The RestroPress - Menu Cart plugin is useful for the RestroPress users who want to show a cart menu on the header.

Available features:
- You can chnage the cart icon image by choose available icon option.
- Setting option provided to show cart quantity with cart total price.
- Manage to show the cart icon as per the cart value is empty or not.

Note:- To view the cart menu, you must have a Primery menu. If you have not created any primery menu then Go through this setting path Appearance -> Menu -> Here you need to create a new menu and set that as Primery menu.

== Installation ==
1. Login to your WordPress dashboard and navigate to Plugins > Add New
2. Search for "RestroPress - Menu Cart".
3. Click install.
4. Click activate.
5. Once the plugin is installed then you can see RestroPress on the left navigation bar of WordPress Dashboard.
